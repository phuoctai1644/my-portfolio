## Run locally
### 1. Clone respository
```
$ git clone https://github.com/phuoctai1644/my-portfolio.git 
$ cd my-portfolio
```
### 2. Run the app
```
$ npm install
$ npm start
```
The app will run in the development mode.
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
